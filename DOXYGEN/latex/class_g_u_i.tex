\doxysubsection{Detailed Description}
Manages all \doxylink{class_g_u_i}{GUI} elements in the game including main menu, game over, map, and battle screens. 

This class is responsible for creating and managing all the HUDs used in the game. It provides methods to access different screens and handles their initialization.
