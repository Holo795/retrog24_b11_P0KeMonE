\doxysubsection{Detailed Description}
Manages database operations for Pokémon. 

This class encapsulates all database interactions in a Pokémon game, including retrieving and storing Pokémon and their moves.

Retrieves a random Pokémon from the database. 

Fetches a random Pokémon from the database and returns a pointer to a newly created Pokémon object.

\begin{DoxyReturn}{Returns}
A new Pokémon object initialized with the fetched data.

This function executes a SQL query to select a random Pokémon from the database. It retrieves the Pokémon\textquotesingle{}s attributes and creates a new Pokémon object with those attributes. If no moves are associated with the Pokémon, the function recursively fetches another random Pokémon.

\begin{DoxyReturn}{Returns}
A pointer to a newly created Pokémon object, or nullptr if the query fails or no results are returned. 
\end{DoxyReturn}
